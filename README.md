# NLP-project-baseline
baseline for nlp-project

## 데이터 전처리

- 데이터 전처리는 pipeline으로 처리한다.
- dataloader를 사용한다.
- 데이터는 parquet를 기본형식으로 하며 샘플 csv를 같이 저장한다.
